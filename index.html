Examples of Data Analtyics and Mining

Big Data is everywhere it seems today, but has'nt there always been a lot of data for some businesses? The real issue is that it is now Easy Data. You can gather so much data from private and public sources that it can be easily overwhelming. The focus of most firms is to figure to how to automate the analysis of this data by building ever more elaborate algorithms and models. However, the one thing you cannot build into a model is human intuition. No computer can be taught the domain knowledge that you and your people have. Listed below are several examples of data analysis techniques and outputs, but only you can see how it relates to your business.
